## 4. 安全测试结果

| 安全测试项 | 测试结果 | 说明 |
|------------|----------|------|
| SQL注入 | 通过 | SQLAlchemy ORM参数化查询,注入攻击返回业务错误 |
| XSS攻击 | 通过 | 后端安全存储,Vue3模板自动转义,脚本不执行 |
| CSRF | 通过(不适用) | 使用JWT Bearer Token(非Cookie),CSRF攻击面不存在 |
| 认证授权 | 通过 | 普通token无法访问admin(403),无token返回401 |
| 越权操作 | 通过 | A用户无法操作B用户数据,接口校验buyer_id/seller_id |
| 密码存储 | 通过 | Werkzeug pbkdf2:sha256哈希,非明文 |
| 重复操作防护 | 通过 | 评价/举报UniqueConstraint,商品重复购买校验 |
