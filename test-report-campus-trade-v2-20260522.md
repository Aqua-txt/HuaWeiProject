## 5. 安全测试结果

| 测试项 | 结果 | 说明 |
|--------|------|------|
| SQL注入 | 通过 | SQLAlchemy ORM参数化查询 |
| XSS | 通过 | Vue3自动转义+后端安全存储 |
| CSRF | 不适用 | JWT Bearer Token(非Cookie),无CSRF风险 |
| 认证授权 | 通过 | 普通token->admin接口返回403 |
| 越权操作 | 通过 | 接口校验buyer_id/seller_id |
| 密码存储 | 通过 | Werkzeug哈希(pbkdf2:sha256) |
